# TooManySuits

Installation
-

- Install BepInEx.
- Download and install More Suits.
- Download and install TooManySuits.

Default Controls
-

- Next Page: n
- Back Page: b

These are configurable in the config, located at Lethal Company\BepInEx\config\verity.TooManySuits.cfg
